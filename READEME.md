To run your project, navigate to the directory and run one of the following pnpm commands.

- cd apps/mobile
- pnpm run android
- pnpm run ios
- pnpm run web

# Start backend

pnpm --filter backend dev

or

cd apps/backend
pnpm dev

# Start expo mobile app

pnpm --filter mobile start

or

cd apps/mobile
npx expo start --web
